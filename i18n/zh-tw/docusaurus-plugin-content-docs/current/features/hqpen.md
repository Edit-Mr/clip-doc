## 如何使用

1. 點選選單中的「其他」。
2. 點選「設定」
3. 找到「播放器」->「高清畫筆」, 設定它。
4. 生效!

## 注意
1. 我們計劃於 ClipCC 3.2 正式發佈此功能。
2. 該功能可能會影響專案的渲染效率。
